This directory contains the parts modified for easier printing on  all home grade printers.
Parts are a bit heavier than the original but they should withstand more mechanical stress.

Parts list:

RW_ClawLeft_Heavy.stl ........... Left Claw .......... 3x
RW_ClawRight_Heavy.stl .......... Right Claw ......... 3x
RW_CrankD345L8.stl .............. Crank .............. 2x
RW_FrameBR.stl .................. Right frame ........ 2x
RW_FrameBL.stl .................. Left frame ......... 2x

Sources:

RW_Claw_original.stl ............ Original claw
RW_Claws.scad ................... OpenSCAD source for the claws
RW_CrankD345L8.mix .............. Meshmixer source of the crank
RW_FrameBL.mix .................. Meshmixer source of the left frame
RW_FrameBL_original.stl ......... Original left frame
RW_FrameBL_shaft.scad ........... OpenSCAD source of the left frame shaft
RW_FrameBL_shaft.stl ............ STL left frame shaft (user for the meshmixing)
RW_FrameBR.mix .................. Meshmixer source of the right frame
RW_FrameBR_original.stl ......... Original left frame
RW_FrameBR_shaft.scad ........... OpenSCAD source of the right frame shaft
RW_FrameBR_shaft.stl ............ STL right frame shaft (user for the meshmixing)
